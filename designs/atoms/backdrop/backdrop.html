Token recipe

Property Token / Value Dark resolved Light resolved Notes
background (--scrim) rgba(0,0,0,0.55) — hardware-black exception rgba(0,0,0,0.55) rgba(0,0,0,0.55) Theme-invariant. See README § Hardware-black.
background (--scrim-light) rgba(0,0,0,0.30) — hardware-black exception rgba(0,0,0,0.30) rgba(0,0,0,0.30) Lighter dim for popovers/menus.
background (--blur) rgba(0,0,0,0.40) — hardware-black exception rgba(0,0,0,0.40) rgba(0,0,0,0.40) Bottom sheet frosted.
background (--blur-strong) rgba(0,0,0,0.50) — hardware-black exception rgba(0,0,0,0.50) rgba(0,0,0,0.50) Full-screen plan-review.
backdrop-filter (--blur) blur(8px) — structural geometry blur(8px) blur(8px) -webkit- prefixed for Safari.
backdrop-filter (--blur-strong) blur(20px) — structural geometry blur(20px) blur(20px) -webkit- prefixed for Safari.
transition-duration --motion-medium 220ms 220ms Fade in/out duration.
transition-timing-function --motion-ease-out cubic-bezier(0.16,1,0.3,1) cubic-bezier(0.16,1,0.3,1) Spring-like deceleration.
animation-duration (is-entering / is-exiting) --motion-medium 220ms 220ms Keyframe animation duration.
z-index 100 — structural layer 100 100 Modal/sheet content renders at z-index 60+ above (in context).

TOKEN_GAPs: none. rgba(0,0,0,X) background values are hardware-black structural exceptions (same precedent as Dynamic Island + device bezel). Blur radii 8px/20px are structural geometry (same exception category as the 6px dot in live-activity-dot).
